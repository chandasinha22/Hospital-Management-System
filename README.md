# Hospital Management System
A Java + MySQL CRUD application to manage patient records.

## Features
- Add Patient
- View All Patients  
- Delete Patient

## Tech Stack
- Java
- MySQL (XAMPP/MariaDB)
- JDBC Connector

## How to Run
1. Start XAMPP MySQL on port 3307
2. Create database: `hospital_db`
3. Run: `java -cp "out;lib\mysql-connector-j-9.7.0.jar" main`
